Smart Sign Language Translation System

Project Overview

This project aims to develop a real-time sign language translation system capable of recognizing hand gestures from video input and converting them into readable text. The objective is to improve communication between hearing-impaired individuals and non-sign-language users.

Project Goals

* Capture sign language gestures from a camera.
* Process video frames using machine learning techniques.
* Convert recognized signs into text output.
* Deploy the application on an embedded Linux platform built using Buildroot.

Hardware Platform

* Laptop development environment
* USB Camera
* Embedded Linux target (QEMU for initial testing)

Build System

Buildroot will be used to generate the target Linux image and required packages.
